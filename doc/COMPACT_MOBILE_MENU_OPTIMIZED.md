
### **✅ КОМПАКТНЫЕ ОТСТУПЫ:**
```css
/* БЫЛО → СТАЛО */
mb-8  → mb-6     /* Отступы блоков */
mb-12 → mb-8     /* Отступы навигации */
mb-8  → mb-6     /* Отступ кнопки */
space-y-6 → space-y-3  /* Расстояния между ссылками */
space-y-6 → space-y-4  /* Расстояния соцсетей */
```

### **✅ УМЕНЬШЕННЫЕ РАЗМЕРЫ:**
```css
/* ТЕКСТ */

/* КНОПКИ */
px-12 py-4 → px-8 py-3   /* CTA кнопка */
```
